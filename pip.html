Installer pip

(Windows)Installer pip

  1. Veuillez désisntaller Python de la même manière qu'un quelquonque autre logiciel
  2. Télécharger le setup de Python adapté à votre version de Windows ici
  3. Lors du démarrage du setup (en accoradant les droits d'administrateur) avant de continuer l'installation cocher "Ajouter PythonX.X dans le PATH"(voir image ci-dessous), puis continuer l'installation
  4. Terminer l'installation comme une installation normale
  5. Une fois l'installation terminé REDÉMARRER votre ordinateur!
  6. pip est opérationnel

(Linux)Installer pip

  1. Ouvrez le terminal
  2. Entrez sudo apt install python3-pip puis validez
  3. Entrez votre mot de passe d'utilisateur (durant l'écriture du mot de passe sur Ubuntu, le curseur ne bouge pas pour cacher votre mot de passe)
  4. Patientez...
  5. pip est installé

(Tout système confondus)Installer virtualenv

Virtualenv sert à créer des environnements virtuels Python (et donc inclure les modules Python non-standarts)

  1. Ouvrez l'invite de commande ou le terminal
  2. Entré "pip install virtualenv"
  3. Virtualenv est installé

(Tout système confondus)Installer des modules (exemple ici: keyboard)

Installer des modules python avec pip (exemple avec le module keyboard pour Jours Frais)

  1. Ouvrez le terminal ou l'invite de commande (pour ouvrir l'invite de commande, faites Windows+R puis entré cmd.exe et validez)
  2. (Sur Windows)Taper pip install keyboard et validez!
    (Sur Linux) Taper pip3 install keyboard et validez!
  3. Keyboard est installé

Pour installer un tout autre module, remplacez keyboard par le nom du module que vous voulez installer!
