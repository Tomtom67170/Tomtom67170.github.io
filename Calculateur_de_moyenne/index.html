Calculateur de moyenne

C'est quoi Calc... enfin ça! Version 2

Calculateur de moyenne est un logiciel qui calculer une moyenne à l'aide d'autre moyenne (donc calculer une moyenne générale)

Ça marche comment

Lors du démarrage (en plus de vous demander si vous voulez voir un tutoriel qui est ici) vous devrez choisir un fichier .csv qui contiendra vos notes. Ce fichier .csv, c'est une extension Chrome qui vous le créeras!
Une fois le fichier .csv chargé, le logiciel s'occupe de calculer votre moyenne puis vous l'affiche dans une autre fenêtre

Et la V 1.0

Elle est compatible avec tous les naviguateurs mais demande beaucoup plus de main d'oeuvre de la part de l'utilisateur
En effet l'utilisateur doit entrer les moyennes lui-même!
Recommandé si vous n'avez pas Chrome pour PC et Mac
