Pourquoi Android n'est pas compatible (et ne le sera jamais) avec Jours Frais?

Android (comme beaucoup de systèmes d'exploitation) ne lit pas par défault le Python

Android est un système d'exploitation qui comme beaucoup ne peut pas lire le Python!
Cependant des applications sont disponibles pour éxécuter le language Python (Pydroid3)!

Mais alors pourquoi pas Android?

À cause du MULTITACHES!
Android n'est pas un système gérant le multitache notamment en Python
Jours Frais utilise le fonction python "os.startfile()" cette fonction a pour but de démarrer des fichiers comme si on cliquez dessus dans l'explorateur (Linux utilise une autre fonction car "os.startfile()" ne fonctionne pas).
Si Jours Frais utilise cette fonction c'est pour mettre en marche E1.py. Cette extension a pour but de gérer le bon fonctionnement de Jours Frais sans cette extension, Jours Frais ne fonctionnera pas correctement.

À partir de la 0.2 Jours Frais vérifieras si il tourne sur Android et le différencieras de Linux
