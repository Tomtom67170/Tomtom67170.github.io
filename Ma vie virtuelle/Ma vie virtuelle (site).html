C'est quoi Jours Frais?

Jours Frais est jeu ou vous vous controlez dans une autre vie et cela en gérant 4 éléments (L'énergie, la faim, l'humeur et l'hygiène).
Pour garder ces éléments au max de leur "forme", vous pourrez vous aidez des pièces de votre maison. Cette maison vous à été "donné" par John Business, votre patron! John Business, fondateur de Johnservice, vous donnant une maison, vous aurez des dettes à payer

Quels sont les platformes de Jours Frais?

Scratch

Le developpement de cette version est actuellement suspendu!

Le jeu se nomme toujours Ma vie virtuelle

Jours Frais (sous le nom de Ma vie virtuelle) est disponible sur Scratch 3.0 mais il est disponible sur la version en ligne de Scratch
Cette version possède une interface graphique assez complet

Python

Jours Frais a aussi vu le jour en Python3 pour reprendre le concept de la version Scratch mais en une version "executable" et plus simple d'utilisation!
Cette version à une interface graphique quasi inexistante et ce résumme à un invite de commande (ou au terminal chez Unix)

Quel est le but de Jours Frais?

Le but de Jours Frais est de gérée 4 éléments (énergie, faim, humeur, hygiène) et de payer un montant de 500 000$ pour rembourser John Business.
Pour augmenter vos stats des 4 éléments, vous devrez utilisez différents objets présent dans différentes pièces de votre maison! Pour cela, vous devrez trailler sur un PC, votre PC de travail (le pc est sous Windows XP sur Scratch et est sous MS-DOS6.22 sur Python3)
Vous devrez également cuisiner, gérer votre maison, faire le ménage, bref gérez votre vie!

Installation des modules

Jours Frais utilise plusieurs modules. La pluparts sont installés par défault mais Jours Frais utilise également un module sous le nom de KEYBOARD!
La particularité de ce module: Il n'est pas installé par défault et il faut l'installer manuellement!
Pour cela, une page est disponible pour vous!
