Bot Bing

C'est quoi Bot Bing

Bot Bing est un logiciel d'automatisation vous permettant de automatiquement gagner des rewards pour votre compte Microsoft
Tout est automatique, vous devez juste entrer vos identifiants (vous pouvez choisir de les rentrer vous même sur le site de Microsoft ou alors de laissez le logiciel les entrer à votre place!)
A noter que Tomservice ne récupère aucune informations de connexion ni aucune autre informations! Toutes les informations du bot sont supprimés à chaque fermeture du logiciel
Ce bot va effectuer des recherches sur Bing toutes les 5 secondes sur le compte Microsoft auquel il est connecté!
Le but est d'ainsi gagner des rewards pendant que vous ne faites rien!
ATTENTION! Microsoft Rewards a fixé une limite de 90 points (soit 30 recherches) pouvant être gagné par jour pour des recherches sur PC(plus 12 points bonus sur Edge)!
Le bot ne s'arrêtera pas au bout des 30 recherches, nous vous invitons à le fermer manuellement en fermant d'abords le naviguateur puis la console!
Nous ne sommes pas tenu responssable de tous dommages provoqué par le bot sur votre machine ou compte Microsoft!

Informations importantes concernant les modules

Ce logiciel comporte des modules non-standarts a Python3, leurs installation est obligatoire pour utiliser le logiciel(a l'exception de l'executable Windows)!
Un installateur de module est disponible ici

L'installateur est encore en devellopement!
Pour les connaisseurs, les modules selenium, webdriver_manager et package peuvent être installé avec pip/pip3!

Comptabilité

Naviguateur

Le logiciel a été programmé et conçu pour les naviguateurs Google Chrome et Microsoft Edge
Un support pour Mozilla Firefox et Safari est envisagé
La prise en charge du naviguateur Opera avait été envisagée cependant selenium en a arreté le support dans une récente mise a jour!
Les autres naviguateurs non cité précédemment ne sont pas prévue pour le développement futur du bot
